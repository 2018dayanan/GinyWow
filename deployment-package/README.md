## Quick Start (बस 5 Minutes में!)

### 1. Extract Files
```bash
tar -xzf ginywow-website-complete.tar.gz
cd ginywow-website-complete
```

### 2. Start Website (One Command!)
```bash
./start-server.sh
```

### 3. Visit Your Website
Open browser: `http://localhost:3000`
